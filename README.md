# Raspberry Pi LED Control 
A Python-based LED control system using Raspberry Pi GPIO,
featuring automated LED blinking with precise timing control.

## Components Used
- Raspberry Pi Pico
- LED (Red)
- Resistor (220Ω)
- Breadboard
- Jumper Wires

## Circuit Connection
- GP13 → Resistor → LED (+)
- GND → LED (-)

## How It Works
- LED turns ON for 1 second
- LED turns OFF for 1 second
- This cycle repeats continuously

## Tools Used
- Wokwi Simulator
- MicroPython
